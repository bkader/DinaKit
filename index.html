Grid System

A grid is used to harmonize negative space in layout

Introduction

A grid is used to design layouts and page parts. It is used to align negative space in designs.

Using a grid makes content appear to flow more naturally on site pages.

Container

The container centers all elements on the page: <div class="container">...</div>

The container may contain as many row as you desire.


Rows

Rows are used to divided page parts horizontally. Each row may contain as many columns as you want. To use a row, simply use: <div class="row">...</div>


Columns

Dinakit uses 12 columns grid. <div class="col-[x|s|m|l]-[1-12]">...</div>

col-x-.. are used for devices which max-width: 767px.

col-s-.. are used for devices which min-width: 768px and max-width: 991px.

col-m-.. are used for devices which min-width: 992px and max-width: 1199px.

col-l-.. are used for devices which min-width: 1200px.

If you want to make your design responsive, make sure to start with mobile first: -x- up to -s-

Breadcrumbs

Everyone (or almost) knows what breacrumbs are! In case you don't, it's the element that lets you know on what page your are and where you came from. No matter how I explain it, showing it to you is better:

Home
Category
Post Title

Pagination and Pager

What's the difference between these a pagination and a pager?

In fact, they are almost the same! The only difference is that in paginations we display pages numbers and we can display or not items with previous and next pages.

Color Variations

By adding .progress-[color] from purple to white to <progress...> element you change the color.

As for .progress elements, the color is applied to the .fill child like so: .fill.fill-[color].


Text Value

In order to display the text inside the progress bar <progress> element should be droped and a <span class="progress">...</span> should be used. Examples:

<div class="progress">
    <div class="fill" style="width: 35%" data-fill="35"></div>
    <div class="text"></div>
</div>
