A grid is used to harmonize negative space in layout

Using a grid makes content appear to flow more naturally on site pages.

Container

The container centers all elements on the page: <div class="container">...</div>

The container may contain as many row as you desire.


Rows

Rows are used to divided page parts horizontally. Each row may contain as many columns as you want. To use a row, simply use: <div class="row">...</div>


Columns

Dinakit uses 12 columns grid. <div class="col-[xs|sm|md|lg|xl]-[1-12]">...</div>

col-x-.. are used for devices which max-width: 767px.

col-s-.. are used for devices which min-width: 768px and max-width: 991px.

col-m-.. are used for devices which min-width: 992px and max-width: 1199px.

col-l-.. are used for devices which min-width: 1200px.

If you want to make your design responsive, make sure to start with mobile first: -x- up to -s-

Buttons

A button is a nicer way to display HTML buttons and links and indicates a possible user action.


Types

Simple

A standard button is used by adding the class btn to a button or any other element.

a tag

<a href="#" class="btn">...</a>
<button class="btn">...</button>
<input type="..." class="btn">


Colored Buttons

You can colorize a button by adding available colors: btn-purple ... Available colors are: purple, blue, teal, green, orange, yellow, red, brown, pink, black and white.

Hollow Buttons

Hollow buttons are outlined buttons with transparent background and colored borders. Hollow buttons are available for all colors. Simple add btn-hollow and you're done!
